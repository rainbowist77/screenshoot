# ScreenShoot
A simple screenshot tool allowing users to save screenshots and paste them into clipboard.
## Introduction
This is my first Java program using Maven, aiming to avoid poor screenshoot problems with Windows or QQ.
## Features
All you need is to press F9, and then your screenshot will automatically appear both on your PC and clipborad.
## Usage
Please locate the system tray first, and then:
- left-click to open the latest file
- right-click to open the popup menu
## FAQ
### Q: How can I change the hotkey?
A: To simplify things, I have not added a config.properties file. To achieve that, please modify the source code and package it yourself.
